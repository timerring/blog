The Tips of Macos Automator

The Automator is a powerful tool for automating tasks on macOS. It allows you to create custom workflows that can be triggered by various events, such as opening a file, pasting text, or receiving an email. This article will share my common usages of Automator.

Location #

For the Quick Action, you can find the Automator in the ~/Library/Services folder.

cd 	~/Library/Services

Quick Action #

Attention: The type is Quick Action. Not Application or Workflow.

New iTerm Window #

Apple Scripts:

tell application "iTerm"
  reopen
  activate
end tell

Then you can set the Shortcuts:

System Settings -> Keyboard Shortcuts -> Services -> General.
